## 1. 项目概述

本系统用于监控单个设备房的温湿度，并通过值班室远程控制风扇。系统采用一主一从架构，值班室放置**主机**，设备房配置一台**从机**，通过 LoRa 无线模块实现 300m 以上的稳定通信。

### 核心功能

| 功能 | 说明 |
|------|------|
| 温度采集 | DS18B20 防水探头，精度 ±0.5°C |
| 湿度采集 | DHT22 (AM2302)，精度 ±2% RH |
| 无线通信 | LoRa SX1278 @ 433MHz，穿墙 ≥ 300m |
| 自动温控 | 温度超过阈值自动启动风扇，带回差控制 |
| 远程控制 | 主机按键控制风扇开/关、模式切换、阈值调整 |
| 状态显示 | P10 点阵 LED 屏显示实时数据 |
| 离线检测 | 10 秒无数据上报显示"离线"告警 |

## 6. 通信协议

### 6.1 帧格式

```
┌──────┬──────┬──────┬──────┬──────┬──────────┬──────┬──────┐
│ HEAD │ SRC  │ DST  │ TYPE │ LEN  │   DATA   │ XOR  │ TAIL │
│ 0xAA │ 1B   │ 1B   │ 1B   │ 1B   │ 0~16B    │ 1B   │ 0x55 │
└──────┴──────┴──────┴──────┴──────┴──────────┴──────┴──────┘
  总长度: 7 + DATA_LEN 字节 (最大 23 字节)
  异或校验范围: SRC 到 DATA 结尾
```

### 6.2 设备 ID

| ID | 设备 |
|----|------|
| 0x00 | 值班室主机 (Master) |
| 0x01 | 设备房从机 (Slave) |

### 6.3 指令类型

| 类型值 | 名称 | 方向 | 数据内容 |
|--------|------|------|----------|
| 0x01 | CMD_FAN_CONTROL | 主→从 | 0x01=开风扇, 0x00=关风扇 |
| 0x02 | CMD_SET_THRESHOLD | 主→从 | 温度阈值 20~50 (1字节, 单位°C) |
| 0x03 | CMD_SET_MODE | 主→从 | 0x01=自动模式, 0x00=手动模式 |
| 0x11 | RPT_SENSOR_DATA | 从→主 | 传感器数据 (7字节, 见下) |

### 6.4 RPT_SENSOR_DATA 数据结构

```
偏移  长度  字段                说明
─────────────────────────────────────────────
0     2     temperature_x10    温度 ×10 (大端序, 如 285 = 28.5°C)
2     2     humidity_x10       湿度 ×10 (大端序, 如 650 = 65.0%)
4     1     fan_status         0x01=开, 0x00=关
5     1     mode               0x01=自动, 0x00=手动
6     1     threshold          温度阈值 (20~50)
```

### 6.5 通信时序

```
[从机]                            [主机]
   │                                │
   │──── RPT_SENSOR_DATA (3s周期) ──►│  上报温湿度数据
   │                                │
   │◄─── CMD_FAN_CONTROL ──────────│  用户按键触发
   │◄─── CMD_SET_THRESHOLD ────────│  用户调整阈值
   │◄─── CMD_SET_MODE ─────────────│  用户切换模式
   │                                │
   │  ... 10s 无数据 ...            │
   │                                │──► 显示"离线"
```

## 7. 软件架构

### 7.1 分层结构

```
┌─────────────────────────────────────────┐
│              应用层 (Application)         │
│  app_slave.c / app_master.c              │
│  温控逻辑 | 按键处理 | 显示更新 | 状态管理  │
├─────────────────────────────────────────┤
│              驱动层 (Drivers)            │
│  ds18b20.c | dht22.c | relay.c          │
│  hub75.c | key.c | sx1278.c            │
├─────────────────────────────────────────┤
│              协议层 (Protocol)           │
│  protocol.c                              │
│  帧打包 | 帧解析 | 异或校验               │
├─────────────────────────────────────────┤
│              硬件抽象层 (HAL)             │
│  STM32 标准外设库 (StdPeriph)            │
│  GPIO | SPI | SysTick | TIM | EXTI     │
└─────────────────────────────────────────┘
```

### 7.2 从机主循环流程

```
main() → App_Slave_Init() → while(1) → App_Slave_Loop()
                                         │
                        ┌────────────────┤
                        │                │
                  每 2000ms? ─── 是 ──► 启动 DS18B20 转换
                        │                │
                        │          等待 750ms
                        │                │
                        │          读取 DS18B20 温度
                        │                │
                        │          读取 DHT22 湿度
                        │                │
                        │          自动温控判断
                        │           ├── 温度 ≥ 阈值 → 开风扇
                        │           └── 温度 ≤ 阈值-1°C → 关风扇
                        │
                  每 3000ms? ─── 是 ──► LoRa 上报传感器数据
                        │
                   LoRa 接收中断 ──► 解析帧 → 执行控制指令
```

### 7.3 主机主循环流程

```
main() → App_Master_Init() → while(1) → App_Master_Loop()
                                          │
                         ┌────────────────┤
                         │                │
                   每 20ms? ──── 是 ──► 扫描 4 个按键
                         │                │
                         │         KEY1: 手动开/关风扇
                         │         KEY2: 切换自动/手动
                         │         KEY3: 阈值 +1°C
                         │         KEY4: 阈值 -1°C
                         │                │
                         │         发送 LoRa 控制指令
                         │
                  检查超时 ──────────► 10s 无数据 → 标记离线
                         │
                  每 500ms? ──── 是 ──► 更新点阵屏显示

#### sx1278.h / sx1278.c — LoRa 无线驱动

```
功能: SX1278 LoRa 模块 SPI 接口驱动, 支持中断收发

硬件接口: SPI1 (PA4=NSS, PA5=SCK, PA6=MISO, PA7=MOSI)

├─────────────────────────────────────────────────────────────┤
│ SX1278_Send(data, len)                                      │
│   作用: 发送数据 (非阻塞, 通过 DIO0 中断确认完成)             │

LoRa 参数配置:

  扩频因子:   SF9
  带宽:       125 kHz

  发射功率:   17 dBm
  同步字:     0x12
  前导码:     8 符号
```

### 8.2 从机模块 (slave/)

#### ds18b20.h / ds18b20.c — 温度传感器驱动

```
功能: DS18B20 单总线温度传感器驱动 (PA0)

关键函数:
┌─────────────────────────────────────────────────────────────┐
│ DS18B20_Init()                                              │
│   作用: 初始化 GPIO, 检测传感器存在                          │
├─────────────────────────────────────────────────────────────┤
│ DS18B20_StartConvert()                                      │
│   作用: 发送温度转换命令 (非阻塞)                             │
│   协议: Skip ROM (0xCC) + Convert T (0x44)                  │
├─────────────────────────────────────────────────────────────┤
│ DS18B20_ReadTemp()                                          │
│   作用: 读取转换完成的温度值                                  │
│   协议: Skip ROM (0xCC) + Read Scratchpad (0xBE)            │
│   返回: 温度值 (°C), 失败返回 -100.0                         │
├─────────────────────────────────────────────────────────────┤
│ DS18B20_ReadTempBlocking()                                  │
│   作用: 完整读取 (启动转换 + 延时750ms + 读取)               │
│   阻塞时间: 约 750ms (12位精度)                              │
└─────────────────────────────────────────────────────────────┘

单总线时序 (72MHz 主频):
  复位脉冲:  拉低 480μs → 释放 → 等待 60μs 检测存在脉冲
  写 1:      拉低 2μs → 释放 60μs
  写 0:      拉低 60μs → 释放 2μs
  读位:      拉低 2μs → 释放 → 采样 10μs 后
```

#### dht22.h / dht22.c — 温湿度传感器驱动

```
功能: DHT22 (AM2302) 温湿度传感器驱动 (PA1)

关键函数:
┌─────────────────────────────────────────────────────────────┐
│ DHT22_Init()                                                │
│   作用: 初始化 GPIO 为输入 (外部 10kΩ 上拉)                 │
├─────────────────────────────────────────────────────────────┤
│ DHT22_Read(data)                                            │
│   作用: 读取温湿度数据                                       │
│   流程: 发送起始信号 → 检测响应 → 读取 40bit → 校验          │
│   返回: 1=成功, 0=失败                                       │
│   输出: data->temperature (°C), data->humidity (%RH)        │
└─────────────────────────────────────────────────────────────┘

DHT22_Data_t 结构体:
  temperature: 浮点温度值 (支持负温度)
  humidity:    浮点湿度值
  valid:       数据有效性标志

数据格式 (40bit):
  湿度高8位 + 湿度低8位 + 温度高8位 + 温度低8位 + 校验和

## 10. 安装与调试

### 10.1 调试检查清单

| 序号 | 检查项 | 方法 | 预期结果 |
|------|--------|------|----------|
| 1 | LoRa 通信 | 从机上电，主机检查接收 | 主机屏幕显示温湿度数据 |
| 2 | DS18B20 读数 | 手握传感器 | 温度值上升 |
| 3 | DHT22 读数 | 呵气到传感器 | 湿度值上升 |
| 4 | 继电器动作 | 按主机 KEY1 | 从机继电器"嗒"一声，风扇转动 |
| 5 | 自动温控 | 用吹风机加热传感器 | 温度超过阈值后风扇自动启动 |
| 6 | 按键响应 | 依次按 KEY1~KEY4 | 屏幕显示对应变化 |
| 7 | 离线检测 | 断开从机电源 | 10 秒后主机显示"离线" |
| 8 | 通信距离 | 从机移至 300m 外 | 仍能稳定接收数据 |

### 10.2 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| DS18B20 返回 -100°C | 未接上拉电阻 / 虚焊 | 检查 4.7kΩ 上拉电阻 |
| DHT22 读取失败 | 未接上拉电阻 / 时序不准 | 检查 10kΩ 上拉，确认 72MHz 时钟 |
| LoRa 无通信 | 未接天线 / SPI 接线错误 | 必须接天线，检查 SPI 引脚 |
| 继电器不动作 | GPIO 驱动能力不足 | 加三极管驱动电路 |
| 点阵屏闪烁 | 刷新率低 | 检查 TIM2 配置 |
| PC13 按键不灵敏 | 内部上拉太弱 | 加外部 10kΩ 上拉电阻 |

### 10.3 安全警告

> ⚠️ **220V 强电操作必须断电后进行，由专业电工完成。**
> 
> ⚠️ **LoRa 模块必须连接 433MHz 天线后才能发射，否则可能损坏射频前端。**
> 
> ⚠️ **继电器模块应选用带光耦隔离的型号，确保强弱电安全隔离。**
